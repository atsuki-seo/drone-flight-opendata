# データフォーマット規約

- すべての単位はメートル/秒に統一する
  - 高度: 元データがftの場合は x0.3048 で変換
  - 精度: 元データがftの場合は x0.3048 で変換
- 緯度経度は WGS84 座標系、小数点以下6桁
- タイムスタンプは UTC ISO 8601 形式、秒精度で出力する（ミリ秒以下は切り捨て）
  - 例: `2024-09-25T01:34:54Z`
  - DJI SRT のタイムスタンプはローカル時刻（JST、タイムゾーン情報なし）であり、撮影地のタイムゾーンオフセットを適用して UTC に変換する（日本: -9時間）
- 欠損値は `null` を設定する（空文字列は不可）
  - CSV の空セルは pandas の NaN として読み込まれ、JSON 出力時に `null` に変換する
- 空配列は `[]` を設定する
- JSON出力は `ensure_ascii=False`, `indent=2`
